# ConwayGame
Conway's Game of Life

操作方式:
游戏开始前可以先用鼠标，预设活细胞的位置
鼠标左键按下，对应方块变黑，对应细胞为活细胞；
鼠标右键按下，对应方块变白，对应位置如有细胞，变为死细胞，恢复为空白位置；
s键：保存当前的状态；可以将这次的细胞状态读到txt存档里，下次可以读出来接着运行
l键：加载存档；
按下回车/Space，开始游戏
注意：如果键盘响应不了，切换为纯英文输入法
